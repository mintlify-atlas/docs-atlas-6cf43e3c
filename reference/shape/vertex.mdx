---
title: "Custom Shapes with Vertex"
description: "Build arbitrary 2D and 3D shapes by specifying vertices between beginShape() and endShape(), with support for Bézier, quadratic, and Catmull-Rom curve segments."
---

The vertex API lets you define shapes with any number of points and curve types. The basic pattern is:

1. Call `beginShape()` to open a new shape.
2. Add points with `vertex()`, `bezierVertex()`, `quadraticVertex()`, or `curveVertex()`.
3. Call `endShape()` to finish the shape (optionally passing `CLOSE` to connect the last vertex back to the first).

<Warning>
Transforms (`translate()`, `rotate()`, `scale()`) and other shape primitives (`ellipse()`, `rect()`, etc.) cannot be called between `beginShape()` and `endShape()`.
</Warning>

---

## beginShape

Opens a new custom shape.

The optional `kind` parameter restricts how vertices are connected:

| Kind | Effect |
| --- | --- |
| (omitted) | Arbitrary polygon — all vertices connected |
| `POINTS` | Each vertex is drawn as a separate point |
| `LINES` | Vertices are connected in successive pairs |
| `TRIANGLES` | Every three vertices form a separate triangle |
| `TRIANGLE_FAN` | Triangles share the first vertex (fan pattern) |
| `TRIANGLE_STRIP` | Consecutive triangles share an edge |
| `QUADS` | Every four vertices form a separate quadrilateral |
| `QUAD_STRIP` | Adjacent edges form successive quads |
| `TESS` | Explicit tessellation (WebGL only) |

**Signature**

```js
beginShape([kind])
```

<ParamField path="kind" type="Constant">
  Shape primitive type: `POINTS`, `LINES`, `TRIANGLES`, `TRIANGLE_FAN`, `TRIANGLE_STRIP`, `QUADS`, `QUAD_STRIP`, or `TESS`. Omit for a free-form polygon.
</ParamField>

**Example**

```js
function setup() {
  createCanvas(100, 100);
  background(200);

  // Free-form closed polygon
  beginShape();
  vertex(30, 20);
  vertex(85, 20);
  vertex(85, 75);
  vertex(30, 75);
  endShape(CLOSE);
}
```

## vertex

Adds a straight-line vertex to a custom shape.

Must be called between `beginShape()` and `endShape()`.

**Signatures**

```js
vertex(x, y)
vertex(x, y, z)              // WebGL mode
vertex(x, y, u, v)           // 2D with texture coordinates
vertex(x, y, z, u, v)        // WebGL with texture coordinates
```

<ParamField path="x" type="Number" required>
  x-coordinate of the vertex.
</ParamField>

<ParamField path="y" type="Number" required>
  y-coordinate of the vertex.
</ParamField>

<ParamField path="z" type="Number">
  z-coordinate of the vertex (WebGL mode only). Defaults to `0`.
</ParamField>

<ParamField path="u" type="Number">
  Horizontal texture coordinate for this vertex.
</ParamField>

<ParamField path="v" type="Number">
  Vertical texture coordinate for this vertex.
</ParamField>

**Example**

```js
function setup() {
  createCanvas(100, 100);
  background(200);

  // Draw two separate triangles using TRIANGLES mode
  beginShape(TRIANGLES);

  vertex(30, 75);
  vertex(40, 20);
  vertex(50, 75);

  vertex(60, 20);
  vertex(70, 75);
  vertex(80, 20);

  endShape();
}
```

## endShape

Stops adding vertices and renders the shape.

Pass `CLOSE` to automatically connect the last vertex back to the first, closing the shape.

**Signature**

```js
endShape([mode])
```

<ParamField path="mode" type="Constant">
  Pass `CLOSE` to connect the last vertex to the first vertex. Omit to leave the shape open.
</ParamField>

**Example**

```js
function setup() {
  createCanvas(200, 100);
  background(240);
  noFill();
  stroke(0);

  // Open shape (left)
  beginShape();
  vertex(20, 20);
  vertex(80, 20);
  vertex(80, 80);
  endShape();

  // Closed shape (right)
  beginShape();
  vertex(120, 20);
  vertex(180, 20);
  vertex(180, 80);
  endShape(CLOSE);
}
```

## bezierVertex

Adds a cubic Bézier curve segment to a custom shape.

The curve starts at the **previous vertex** (which acts as the first anchor point). You supply two control points and the final anchor point. Must be called after at least one `vertex()` call inside `beginShape()`.

<Note>
`bezierVertex()` has no effect when a `kind` argument was passed to `beginShape()`.
</Note>

**Signatures**

```js
bezierVertex(x2, y2, x3, y3, x4, y4)
bezierVertex(x2, y2, z2, x3, y3, z3, x4, y4, z4)   // WebGL mode
```

<ParamField path="x2" type="Number" required>
  x-coordinate of the first control point.
</ParamField>

<ParamField path="y2" type="Number" required>
  y-coordinate of the first control point.
</ParamField>

<ParamField path="x3" type="Number" required>
  x-coordinate of the second control point.
</ParamField>

<ParamField path="y3" type="Number" required>
  y-coordinate of the second control point.
</ParamField>

<ParamField path="x4" type="Number" required>
  x-coordinate of the anchor point (end of this segment).
</ParamField>

<ParamField path="y4" type="Number" required>
  y-coordinate of the anchor point (end of this segment).
</ParamField>

**Example**

```js
function setup() {
  createCanvas(100, 100);
  background(200);

  // A crescent moon using two Bézier segments
  beginShape();
  vertex(30, 20);
  bezierVertex(80, 0, 80, 75, 30, 75);
  bezierVertex(50, 80, 60, 25, 30, 20);
  endShape();
}
```

## quadraticVertex

Adds a quadratic Bézier curve segment to a custom shape.

A quadratic Bézier has one control point instead of two. The curve starts at the **previous vertex** and ends at `(x3, y3)`.

**Signatures**

```js
quadraticVertex(cx, cy, x3, y3)
quadraticVertex(cx, cy, cz, x3, y3, z3)   // WebGL mode
```

<ParamField path="cx" type="Number" required>
  x-coordinate of the control point.
</ParamField>

<ParamField path="cy" type="Number" required>
  y-coordinate of the control point.
</ParamField>

<ParamField path="x3" type="Number" required>
  x-coordinate of the anchor point (end of this segment).
</ParamField>

<ParamField path="y3" type="Number" required>
  y-coordinate of the anchor point (end of this segment).
</ParamField>

**Example**

```js
function setup() {
  createCanvas(100, 100);
  background(200);

  noFill();
  beginShape();
  vertex(20, 80);
  quadraticVertex(50, 10, 80, 80);
  endShape();
}
```

## curveVertex

Adds a Catmull-Rom spline vertex to a custom shape.

At least **four** `curveVertex()` calls are needed to draw a visible segment: the first and last act as control points that shape but don't appear on the curve. Duplicate the first and last points if you want the curve to pass through them.

<Note>
`curveVertex()` has no effect when a `kind` argument was passed to `beginShape()`.
</Note>

**Signatures**

```js
curveVertex(x, y)
curveVertex(x, y, [z])   // WebGL mode
```

<ParamField path="x" type="Number" required>
  x-coordinate of the vertex.
</ParamField>

<ParamField path="y" type="Number" required>
  y-coordinate of the vertex.
</ParamField>

<ParamField path="z" type="Number">
  z-coordinate of the vertex (WebGL mode only).
</ParamField>

**Example**

```js
function setup() {
  createCanvas(100, 100);
  background(200);

  noFill();
  beginShape();

  // First point doubled so the curve starts there
  curveVertex(32, 91);
  curveVertex(32, 91);

  curveVertex(21, 17);
  curveVertex(68, 19);

  // Last point doubled so the curve ends there
  curveVertex(84, 91);
  curveVertex(84, 91);

  endShape();
}
```

## beginContour / endContour

Cut a hole (negative space) inside a flat custom shape.

Place `beginContour()` and `endContour()` between `beginShape()` and `endShape()`. The hole's vertices must wind in the **opposite direction** from the outer shape: if the outer shape is clockwise, the hole must be counter-clockwise.

<Warning>
Transforms and other shape primitives cannot be used between `beginContour()` and `endContour()`.
</Warning>

**Example**

```js
function setup() {
  createCanvas(100, 100);
  background(200);

  beginShape();

  // Outer square, clockwise
  vertex(10, 10);
  vertex(90, 10);
  vertex(90, 90);
  vertex(10, 90);

  // Inner square hole, counter-clockwise
  beginContour();
  vertex(30, 30);
  vertex(30, 70);
  vertex(70, 70);
  vertex(70, 30);
  endContour();

  endShape(CLOSE);
}
```

## Full example — five-pointed star

The following example combines `beginShape()`, `vertex()`, and `endShape(CLOSE)` to draw a star by alternating between outer and inner radius points:

```js
function setup() {
  createCanvas(100, 100);
}

function draw() {
  background(200);

  let cx = 50, cy = 50;
  let outerR = 40, innerR = 17;
  let points = 5;

  fill(255, 200, 0);
  stroke(0);

  beginShape();
  for (let i = 0; i < points * 2; i++) {
    let angle = (TWO_PI / (points * 2)) * i - HALF_PI;
    let r = (i % 2 === 0) ? outerR : innerR;
    vertex(cx + r * cos(angle), cy + r * sin(angle));
  }
  endShape(CLOSE);
}
```
