## Get started in three steps

<Steps>
  <Step title="Include p5.js">
    Add p5.js to your HTML page via the CDN — no installation required.
    ```html index.html
    <script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/1.11.1/p5.min.js"></script>
    ```
  </Step>
  <Step title="Write your sketch">
    Define `setup()` to initialize your canvas and `draw()` to render each frame.
    ```js sketch.js
    function setup() {
      createCanvas(400, 400);
      background(220);
    }

    function draw() {
      if (mouseIsPressed) {
        fill(0);
      } else {
        fill(255);
      }
      circle(mouseX, mouseY, 80);
    }
    ```
  </Step>
  <Step title="Open in a browser">
    Open your HTML file in any modern browser and start drawing. No server needed.
  </Step>
</Steps>

<Tip>
  Try p5.js instantly in the [p5.js Web Editor](https://editor.p5js.org/) — no setup required.
</Tip>
