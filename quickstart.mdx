<Tip>
  Want to skip the setup entirely? Open the [p5.js Web Editor](https://editor.p5js.org/) and start coding in your browser immediately.
</Tip>

## Build your first sketch

<Steps>
  <Step title="Create an HTML file">
    Create a new file called `index.html` on your computer and paste in this template:

    ```html index.html
    <!DOCTYPE html>
    <html>
      <head>
        <meta charset="UTF-8" />
        <title>My p5.js Sketch</title>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/1.11.1/p5.min.js"></script>
      </head>
      <body>
        <script src="sketch.js"></script>
      </body>
    </html>
    ```
  </Step>

  <Step title="Create a sketch file">
    In the same folder, create `sketch.js`. The two core functions are `setup()` — which runs once — and `draw()` — which loops every frame:

    ```js sketch.js
    function setup() {
      createCanvas(400, 400);
      background(220);
    }

    function draw() {
      // Draw a circle wherever the mouse is
      circle(mouseX, mouseY, 60);
    }
    ```
  </Step>

  <Step title="Open in a browser">
    Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge). You should see a gray canvas. Move your mouse over it — circles appear wherever you move.

    <Note>
      If you double-click the file to open it, it may not load local assets (images, fonts, data). For that you'll need a simple local server — see [Installation](/installation) for options.
    </Note>
  </Step>
</Steps>

## Add interactivity

Let's extend the sketch to change behavior based on mouse input:

```js sketch.js
function setup() {
  createCanvas(400, 400);
  background(220);
}

function draw() {
  if (mouseIsPressed) {
    fill(0);       // black when mouse is held
  } else {
    fill(255);     // white otherwise
  }
  noStroke();
  circle(mouseX, mouseY, 60);
}
```

`mouseIsPressed` is `true` whenever any mouse button is held. Try clicking and dragging to paint black circles on the canvas.

## Handle keyboard input

```js sketch.js
function setup() {
  createCanvas(400, 400);
  background(220);
}

function draw() {
  circle(mouseX, mouseY, 40);
}

function keyPressed() {
  if (key === 'c' || key === 'C') {
    background(220); // press C to clear the canvas
  }
}
```
